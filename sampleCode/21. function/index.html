Kerangka Function

function namaFunction(){
return (hasil) }

namaFunction()

    Note :
  1. Deklarasikan function dengan menuliskan "function"
  2. Beri nama function sesuai fungsinya/peruntukannya
  3. Return berupa apa yg dikerjakan oleh function dan yang akan ditampilkan
  4. Menjalankan function dengan memanggil nama function diluar functionnya
  5. Pemberian nilai pada parameter(argument) dilakukan di luar function
  6. Parameter adalah variable yang disimpan dalam kurung ketika membuat function
  7. Argument adalah nilai yang diberikan pada parameter
  8. Parameter dibuat ketika fungsi dibuat
  9. Argument dibuat ketika fungsi akan dijalankan

Contoh :

function penjumlahan(parameter1, parameter2){
var total = paramater1 + parameter2;

return (total)
}

document.writln(penjumlahan(25, 25))
